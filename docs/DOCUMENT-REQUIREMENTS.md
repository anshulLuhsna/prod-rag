## How to Upload Documents

### Option 1: SCP from Windows (PowerShell)

```powershell
# First, create folders on VM (run this SSH'd into VM)
# mkdir -p ~/nifty50-rag/data/documents/{HDFCBANK,ICICIBANK,SBIN,BAJFINANCE,KOTAKBANK}/{annual_reports,quarterly_results}

# Then from Windows PowerShell, upload all at once:
scp -r -i "$env:USERPROFILE\.ssh\id_rsa" `
    "C:\path\to\your\documents\*" `
    dev@34.93.17.227:~/nifty50-rag/data/documents/
```

### Option 2: VS Code Drag & Drop

1. Connect to VM via VS Code Remote SSH
2. Open folder: `~/nifty50-rag/data/documents/`
3. Drag and drop PDFs into the correct folders

### Option 3: Google Cloud Storage

```bash
# Upload to GCS bucket from local
gsutil -m cp -r ./documents/* gs://your-bucket/documents/

# Download to VM
gsutil -m cp -r gs://your-bucket/documents/* ~/nifty50-rag/data/documents/
```

## Document Quality Notes

### What Makes a Good Document

✅ **Good:**
- Official PDF from company IR page
- Clear text (not scanned images)
- Contains financial tables
- Has proper page structure

❌ **Avoid:**
- Low-quality scanned PDFs
- Password-protected files
- Corrupted downloads
- Wrong fiscal year

### Fiscal Year Reference

| Indian FY | Calendar Period | Example |
|-----------|-----------------|---------|
| FY2024 | Apr 2023 - Mar 2024 | Q1 = Apr-Jun 2023 |
| FY2023 | Apr 2022 - Mar 2023 | Q4 = Jan-Mar 2023 |
| FY2022 | Apr 2021 - Mar 2022 | Annual = Full Year |

## Next Steps After Upload

1. **Verify documents are in place:**
   ```bash
   find ~/nifty50-rag/data/documents -name "*.pdf" | wc -l
   ```

2. **Start services:**
   ```bash
   cd ~/nifty50-rag

   ```

3. **Run ingestion:**
   ```bash
   docker compose exec backend python -c "
   from app.ingestion.pipeline import ingest_all_documents
   import asyncio
   asyncio.run(ingest_all_documents())
   "
   ```

4. **Test a query:**
